# NOVARA — Fitness Website

**Novara** is a gym and fitness landing page website designed as an information hub for a fitness center. The website provides clear and structured details about available facilities, membership plans, trainers, and promotional offers, all presented in a modern, clean, and easy-to-navigate layout to help visitors quickly understand what the gym offers.

Live Demo:  
https://ardan2008.github.io/Novara/

---

## Features

- Responsive navbar with hamburger menu
- Hero section with call-to-action
- Services & facilities section
- Membership plans
- Trainers showcase
- Newsletter signup form
- Clean & modern UI

---
